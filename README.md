### How to install colors

$ curl -fLo ~/.vim/colors/jellybeans.vim --create-dirs https://raw.githubusercontent.com/nanotech/jellybeans.vim/master/colors/jellybeans.vim

### How to install plug (vim plugins)

$ curl -fLo ~/.vim/autoload/plug.vim --create-dirs \                                                                                         
    https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim

### Installation of NERDtree https://github.com/scrooloose/nerdtree

$ git clone https://github.com/scrooloose/nerdtree.git ~/.vim/bundle/nerdtree
# on vim run  :helptags ~/.vim/bundle/nerdtree/doc/
# then :help NERDTree.txt
